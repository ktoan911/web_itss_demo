# Audio Files for Pomodoro

Drop your own audio files into these folders. The app expects:

## ambient/ (background nature/environment sounds — looped)

Recommended: 1-3 minute mp3 files, can loop seamlessly.
Expected filenames (used by UI):

- `rain.mp3`
- `forest.mp3`
- `ocean.mp3`
- `cafe.mp3`
- `fire.mp3`
- `wind.mp3`

## music/ (focus music tracks — not looped)

Recommended: 3-10 minute mp3 files, instrumental.
Expected filenames:

- `lofi-1.mp3`
- `lofi-2.mp3`
- `piano-1.mp3`
- `piano-2.mp3`
- `nature-1.mp3`
- `nature-2.mp3`

## Suggested free sources

- https://pixabay.com/music/ (royalty-free, free login)
- https://freesound.org/ (CC licensed, free login)
- https://freemusicarchive.org/

## Notes

- File names must match exactly what UI references.
- If a file is missing, that option in the UI will simply skip when clicked.
- mp3 is recommended; ogg also works in Chrome/Firefox.
